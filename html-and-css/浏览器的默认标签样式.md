Google Chrome浏览器下的标签默认样式


## 浏览器默认样式

```css
html {
    display: block;
}

head {
    display: none;
}

body {
    display: block;
    margin: 8px;
}

div {
    display: block;
}

ul, menu, dir, ol {
    display: block;
    
    /* margin: 1em 0px */
    -webkit-margin-before: 1em;
    -webkit-margin-after: 1em;
    -webkit-margin-start: 0px;
    -webkit-margin-end: 0px;
    
    /* padding-left: 40px */
    -webkit-padding-start: 40px;
}

ul, menu, dir {
    list-style-type: disc;
}

ol {
    list-style-type: decimal;
}

li {
    display: list-item;
    text-align: -webkit-match-parent;
}

h1, h2, h3, h5, h6 {
    display: block;
    font-weight: bold;
    
    /* margin和font-size根据标题等级调整 */
}

```
